# API Gateway

API Gateway is a software which its main purpose is to recieve requests from a client and redirect the client to the appropriate services.
In our project, those who will use the application will be the clients, and their every request will be sent to the API Gateway which will route it to the requested service.

Services that will be handled through the API Gateway includes:

## All Employees
1. Login to the user's account
2. Clocking in and out of shifts.
3. Reporting shift information after clocking out, such as Job Type and corresponding data.
4. Reporting personal progress and achievements.

## Managers Only
1. Generating employees reports.
2. Adding new Employees.
3. Assigning cleaning duties for each employee.
